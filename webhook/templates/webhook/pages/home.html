Credentials Tutorial

Webhook is a tool that allows us to receive notifications of events that occur in our Mercado Pago account. These notifications are sent to a URL that we define.

To be able to receive event notifications from Mercado Pago, we must create a webhook and define the URL where we want the notifications to be sent.

To create a webhook, we must go to the configuration section of our Mercado Pago account and go to the Webhooks section.

When we are in the Webhooks section, we must click on the "Create Webhook" button.

Webhooks Tutorial

In the window that opens, we must define the URL where we want notifications to be sent and select the events from which we want to receive notifications.

After we create the webhook, Mercado Pago will send us a test notification at the URL we defined. This notification will reach us the moment we create the webhook.

Ngrok

Ngrok is a tool that allows you to create secure tunnels to locally expose a web server, so that we can receive responses that come from webhoork.

Crie sua conta na ngrok

Ngrok Site

Install the library in the virtual environment.

Download ngrok

COMMAND: pip install pyngrok

Connect your ngrok account with your computer

COMMAND: ngrok config add-authtoken YOUT-TOKEN

Run ngrok

COMMAND: ngrok http 80

Copy the URL that ngrok generated for you and paste it into the webhook configuration.

Example: https://f0b9-2804-14c-1c1-1c1-1c1-1c1-1c1.ngrok-free.app

After you have configured the webhook, you can test it by making a payment.

When you make a payment, Mercado Pago will send you a notification to the URL you defined.

When you receive the notification, you can see the data that Mercado Pago sent you.

Example:

https://f0b9-2804-14c-1c1-1c1-1c1-1c1-1c1.ngrok-free.app/notification

Start the local server on port 80 to receive notifications on another terminal.

COMMAND: python manage.py runserver 127.0.0.1:80

OBS: have the ngrock server and the django server started to work correctly.
